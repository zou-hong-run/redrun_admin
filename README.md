# redrun-admin

## 描述

**采用nestjs + mysql + typeorm + redis + jwt + mailer + swagger 技术的一个以RBAC权限管理架构的基础架构系统**

## 安装依赖

```bash
$ pnpm install
```

## 运行

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run dev

# production mode
$ pnpm run start:prod
```
